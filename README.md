# School Management System

A full-stack School Management System built using:

## Tech Stack
- Backend: Node.js, Express, Prisma, PostgreSQL (NeonDB)
- Frontend: React, Vite, Tailwind CSS
- Authentication: JWT
- Payments: Razorpay
- Methodology: Agile

## Modules
- Authentication & RBAC
- Student Management
- Teacher Management
- Class & Subject Management
